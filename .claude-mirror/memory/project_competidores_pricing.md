## Cuota tecnología web (W3Techs)
WordPress 42,5%, Shopify ~5,1%, Wix ~4,3%, Squarespace ~2,5%, Webflow ~0,9%

**Why:** El hueco no está en "hacer webs baratas" (commoditizado), sino en productizar "conversión mínima + operación continua" a ARPU suficientemente bajo para ganar al DIY en fricción, pero suficientemente alto para sostener ventas outbound + soporte.

**How to apply:** Nuestro pricing debe estar entre DIY y DFY. El diferencial es outbound + demo personalizada, no el editor.
